Set de Intrucciones

El lenguaje OCUNS posee un set de 16 instrucciones, codificadas por un opcode de un unico caracter hexadecimal (0-F).

Las intrucciones pueden dividirse en 3 grupos con formatos diferenciado (I, II y III). Asi como se cuenta con una variedad de operaciones (con distinta cantidad de registros necesarios) tambien se pueden reconocer varios modos de direccionamientos
